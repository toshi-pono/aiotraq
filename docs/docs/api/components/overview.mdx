## その他のコンポーネント

画像のないコンポーネントも含め、すべてのコンポーネントは以下の通りです：

- [am.write](../am.write) - traQ にメッセージを書き込みます
- [am.image](../am.image) - traQ に画像を送信します
- [am.dataframe](../am.dataframe) - pandas の DataFrame や Series を表として送信します
- [am.pyplot](../am.pyplot) - Matplotlib の figure を画像として送信します
- [am.spinner](../am.spinner) - ローディングスピナーを表示します
- [am.clear](../am.clear) - traQ のメッセージをすべて削除します
- [am.clear_message](../am.clear_message) - 特定のメッセージを削除します
- [am.stamp](../am.stamp) - traQ のメッセージにスタンプを押します
- [am.clear_stamp](../am.clear_stamp) - 特定のスタンプを消します
- [am.clear_all_stamp](../am.clear_all_stamp) - すべてのスタンプを消します
